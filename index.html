REPRODUCTIVE SYSTEM

URINARY SYSTEM

IMMUNE/LYMPHATIC SYSTEM

The immune and lymphatic systems serve as the human body's embarrassing defense system. Only powerful enough to protect them from things like germs or dirt, this system is useless against the infinite number of torture devices that we demons utilize.

SKELETAL SYSTEM

Perhaps the only useful system of the human body, the skeletal system gives human beings their shape and structure. Human bones can be re-purposed as weapons and tools for torment.

INTEGUMENTARY SYSTEM

Serving as the most useless system of the human body, this system is composed of things such as skin, hair, and nails, which would all disintegrate instantly in the fiery pits of the underworld.
